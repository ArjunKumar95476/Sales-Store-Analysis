# 📊 Sales Data Analysis using Python

## 📌 Project Overview

This project analyzes Superstore sales data using Python. The objective is to clean the dataset, perform exploratory data analysis (EDA), visualize business trends, and generate actionable insights for decision-making.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Convert date columns into datetime format
- Create additional time-based features
- Analyze monthly sales and profit
- Compare category and sub-category performance
- Analyze customer segments
- Calculate Sales-to-Profit Ratio
- Visualize insights using charts

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

- Superstore Sales Dataset
- Records: **10,000**
- Features: **21+ columns**

## 📌 Key Insights

- October recorded the highest sales.
- January recorded the lowest sales.
- Technology and Furniture generated the highest sales.
- Office Supplies generated comparatively lower sales.
- Phones delivered the highest profit among sub-categories.
- Home Office customers generated the highest sales and profit.
- Sales-to-Profit ratio was calculated to evaluate profitability.
